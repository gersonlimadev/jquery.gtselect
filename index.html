jQuery gtSelect - is a simple plugin

No mystery, no secret.

See the example:

What to include:

javascript

This plugin need jQuery Library

<script type="text/javascript" src="js/jquery.gtselect.js"></script>
css
<link rel="stylesheet" href="css/jquery.selectgt.css" type="text/css" media="screen" />
html
					<select id="favorite_place" name="favorite_place">
						<option value="">Please select your favorite place</option>
						<option value="sao-paulo">São Paulo</option>
						<option value="rio-de-janeiro">Rio de Janeiro</option>
						<option value="recife">Recife</option>
						<option value="porto-alegre">Porto Alegre</option>
						<option value="salvador">Salvador</option>
					</select>

How to use:

$('select').selectgt();

Params:

					$('select').selectgt({
						speed: 200, 		 // speed dropdown list
						width : 300,		 // width box select
						linkRedirect : false // if true, redirect page using value of the selected option tag
					});
